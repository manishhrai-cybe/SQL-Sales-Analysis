
# SQL Sales Data Analysis Project
This project analyzes sales data using SQL to identify top-performing products, customers, and regions.

## Tools Used
- MySQL
- Excel (for data import/export)

## Key Insights
- The West region generated 34% of total revenue.
- Top 5 products contributed 60% of sales.
- Monthly sales grew by 15% in Q2.

- `sales_analysis.sql` → Main queries
- `sales_data.csv` → Dataset
